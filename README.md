## Usage
```shellscript
> sh bin/installer.sh
```

## Copy
- `prefix + Ctrl+[` でコピーモード開始.
- `Space`で選択開始.
- `Enter`でコピー.
- `prefix + Ctrl+]` or `Ctrl+v`でペースト.
